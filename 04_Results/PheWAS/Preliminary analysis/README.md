**phewas_results_adhd**
* Cohort: All participants from ADHD_SUD_tidy_df.csv, *except:*
* Analysis limited to male *or* female sex assigned at birth due to PheTK limitations. Individuals without male or female sex at birth were excluded.
* Variable of interest: ADHD diagnosis
* n ADHD = 9,809
* n no-ADHD = 39,236

**phewas_results_sud**
* Cohort: Participants from ADHD_SUD_tidy_df with at least one substance use disorder diagnosis.
* Variable of interest: ADHD diagnosis
* n ADHD = 3,233
* n no-ADHD = 7,424

Generated used PheTK. Code [here](https://github.com/morganbaker26/adhd-sud/blob/main/03_Analysis/PheTK%20for%20general%20population%20and%20SUD-enriched%20cohorts.ipynb).
